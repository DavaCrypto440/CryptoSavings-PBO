 CryptoSavings-PBO

Project PBO Tabungan Crypto menggunakan Java.

 Isi Project
- Pertemuan 1 - 10 (implementasi materi PBO)
- GUI: Pertemuan10_JavaGUI.java

 Cara Menjalankan
1. Buka folder project di VS Code
2. Pastikan JDK sudah terinstall
3. Jalankan file Java yang diinginkan (Run)

 Package
Menggunakan package sesuai folder project.
